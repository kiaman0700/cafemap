카카오맵 API 키 설정이 필요해요

  1. developers.kakao.com 접속 → 로그인 → 내 애플리케이션 → 애플리케이션 추가
  2. 만든 앱 클릭 → 앱 키에서 JavaScript 키 복사
  3. 앱 설정 → 플랫폼 → Web 플랫폼 등록에서 사이트 도메인에 http://localhost:5188 (또는 파일을 여는 주소) 추가
  4. 제품 설정 → 카카오맵 → 활성화 설정 ON
  5. index.html 맨 아래 KAKAO_APP_KEY 값에 복사한 JavaScript 키를 붙여넣고 새로고침

키를 넣기 전까지 지도가 표시되지 않습니다.
